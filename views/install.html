Installing MEAN Stack on a MAC

To install the MEAN stack you must first download Node.js and MongoDB from their respective websites, which can be accessed down below:

MongoDB
NodeJS

For installing on a Mac, ensure that you first have Homebrew . If you don't have homebrew installed, open Terminal and run the following command:

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

First Install NodeJS, by downloading it from the website and then running the installer (shown below).

Once installing NodeJS, you must next install MongoDB through your newly installed Homebrew through running the following command

brew install mongodb --with-openssl

After installing MongoDB you're now ready to start working with the MEAN stack. Create your project then once your ready to test your code run the following code in Terminal to start your server:

node server.js
