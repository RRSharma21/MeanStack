What is the MEAN Stack?

The MEAN stack is composed of: Mongo.DB, Express.JS, Angular.JS, & Node.JS, hence the name. The MEAN stack is a software stack built entirely on javascript, and is used to build dynamic web applications.

MEAN Stands for:

MongoDB is a NoSQL Database, meaning unlike most SQL based databases MongoDB does not use a relational schema.

Express is a node.js based web application framework used to better maintain code and functionality on the server side.

Angular is a structural framework used on the client side that allows you to decleare dynamic views for your HTML.

NodeJS is a platform built on Chrome's javascript that can be used to create fast and scalable projects.

Pro's

  • MongoDB Schema-less, Extremelly Fast, NoSQL database
  • AngularJS Less Needed Code, Extends HTML with Directives, Two-way data binding
  • NodeJS Uses JavaScript, Growing Community, NPM, Server-Side Applications with JS
