Designing an App

Designing Hello World

First we must make the folder for our MEAN app, create a directory called Hello and cd into it then enter the following command, and then follow the instructions:

$ npm init

After doing that we must now install express and body-parser. Body parser is used to help parse JSON formatting, it isn't used in this demo, however for larger scaled projects it is required.

$ npm install --save express body-parser

We are now ready to code, open a text-editor of your choice and create the following two files in the root, server.js and index.html

The app variable creates an instance of express, then the following 3 app.use statements below are to access bodyParser, is a request to load the index.html file, and then the last app.use is used to create our localhost.

Open your index.html file and enter the code to create your Hello World program, such as above and save it.

Finally open your terminal enter the following code: nodemon server.js to start and run your local server, then open the browser of your choice enter the following URL: localhost:3000.

You have successfully made your first MEAN app!
